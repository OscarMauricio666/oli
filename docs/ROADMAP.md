# Roadmap - oli

## Objetivo Final

Crear un CLI con LLM que tenga **privilegios de casi administrador**, capaz de:
- Leer y escribir cualquier archivo
- Ejecutar comandos del sistema
- Gestionar proyectos completos
- Interactuar con servicios externos (GitHub, Docker, etc.)

Similar a **Claude Code** pero usando modelos locales (Ollama).
